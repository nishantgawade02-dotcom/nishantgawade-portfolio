ICO Fundraising Success Prediction

University of Leeds

Overview

A machine learning project predicting the fundraising success of 6,146 Initial Coin Offerings (ICOs) using historical blockchain project data. The goal was to identify which project characteristics most reliably predict whether an ICO will hit its funding target.

The Business Question

With thousands of ICOs launching annually and high failure rates, investors and regulators need reliable signals to assess project viability early. This project built a predictive screening tool using historical data to surface the strongest success indicators.

Methodology

  • Dataset: 6,146 ICO projects with 24 features
  • Data cleaning: missing values, outliers, skewed distributions, variable encoding
  • Feature engineering: ICO duration, binary success labels, platform indicators, log-transformed price variables
  • EDA: patterns in team size, investment requirements, whitepaper presence, geographic distributions
  • Models built: Logistic Regression, SVM (RBF kernel), Random Forest, XGBoost
  • Evaluation: 5-fold cross-validation, AUC, Accuracy

The Data

ROC Curve Comparison — XGBoost (AUC: 0.709) leads all models, followed by Random Forest (0.699), Logistic Regression (0.693), and SVM Radial (0.683). Evaluated on 6,146 ICO projects via 5-fold cross-validation.

Key Findings

Finding 01

XGBoost achieved AUC 0.71 — best performing model across all metrics

Outperformed all baseline and ensemble alternatives

Finding 02

Top predictor: Project rating — strongest single signal of ICO success

Highest feature importance score across all models

Finding 03

Team size, ICO duration, and token price are the next most predictive features

Validated via 5-fold cross-validation across all 6,146 ICOs — results are robust, not overfit

Finding 04

Whitepaper presence and GitHub activity signal project credibility

Transparency and technical activity correlate with funding success

Business Implications

These findings provide a practical screening framework for early-stage crypto investment. A project with a high rating, experienced team, reasonable duration, and active GitHub presence is significantly more likely to hit its funding target. Regulators could use similar models to flag high-risk offerings proactively before they reach the market.
